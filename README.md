# 이 겜 해봤음?
http://www.played-this-game.shop/ <br><br>
게임을 추천해 줄 수 있는 사이트를 만들어 보았습니다.
<h2>메인 화면</h2>
<img src="https://user-images.githubusercontent.com/75728238/165766155-ad8b40ab-1d6c-465a-8d82-f8431a0f71e3.PNG" width="700">
<br>
프론트 엔드에서는 타임리프와 자바스크립트 캔버스를 사용하여 꾸며 보았습니다 <br>
<h2>사용 기술</h2>
<img src="https://user-images.githubusercontent.com/75728238/165766167-6ba1bfbe-2a4a-4565-a0be-0c24e481c681.PNG" width="700">
AWS와 도커를 사용하여 배포를 하였으며 스프링부트의 JPA를 사용하여 개발하였으며 <br>
MYSQL의 데이터베이스를 이용하여 사용자에게 게임을 추천해 줍니다.<br>
<h2>정보 입력</h2>
<img src="https://user-images.githubusercontent.com/75728238/165766172-16649a98-3db4-48e9-819f-a97073bc6683.PNG" width="700">
이름을 입력합니다. 이름은 DTO에 저장되어 결과창에 출력합니다. <br>
<h2>설문 조사</h2>
<img src="https://user-images.githubusercontent.com/75728238/165766179-f4108a8b-8c3b-4395-8ab2-b86881e5e012.PNG" width="700">
유저의 성향을 조사합니다. 자바스크립트로 클릭시 자동으로 내려가게 해 주었습니다. <br>
<h2>결과</h2>
<img src="https://user-images.githubusercontent.com/75728238/165766192-96a6f9d7-3c13-4789-aba0-bf6529462746.PNG" width="700">
설문조사를 통해 사용자의 취향에 맞을 게임을 추천해줍니다. 공유버튼을 통해 친구와도 같이 즐길 수 있습니다.
<h2>총평</h2>
AWS와 도커를 사용하여 배포를 해보았으며 도메인을 직접 구매하여 연동을 해보았습니다. <br><br>
도커를 사용 할 수 있는 기회가 되었으며 어떠한 방식으로 스프링부트가 진행되는지 알 수 있었습니다.
